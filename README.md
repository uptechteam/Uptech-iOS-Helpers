# Uptech Helpers 

iOS helper library that contains commonly used extensions in **Uptech** iOS projects.

## What's included
- Array extensions
  - safe subscript
  - next, prev, remove *Element* methods
- UI-in-code helpers
  - UIView extensions:
    - subview adding, insertions *with* constrains
    - corner radius
  - // question. add **all** extensions description? 
- Table View helpers
